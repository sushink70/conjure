┌─────────────────────────────────────────────────────────────┐
│ OPTIONS                                                      │
└─────────────────────────────────────────────────────────────┘

-a {bfs|dfs}      Algorithm (default: bfs)
-s NODE           Starting node (default: auto)
-d SECONDS        Delay per step (default: 0.5)
-t {graph|tree}   Force type (default: auto)

┌─────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING                                              │
└─────────────────────────────────────────────────────────────┘

"Could not find graph" → Use variable name: graph/tree
"Wrong node"           → Use -s to specify start node
"Too fast/slow"        → Adjust with -d SECONDS
"Import error"         → Run: pip install rich
